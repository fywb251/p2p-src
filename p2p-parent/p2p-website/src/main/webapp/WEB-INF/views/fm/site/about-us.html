关于我们

关于我们包括平台介绍、管理团队、资质荣誉、合作伙伴、发展历程、网站公告、网站规则、联系我们八部分内容，具体如下：

一、平台介绍：

和融网（www.herongwangnt.com）是由江苏紫金财富金融信息服务有限公司运营的一家从事互联网金融服务的平台。和融网秉承“诚信立业，稳健行远”的经营理念，通过完善的风控体系和先进的互联网技术的有效结合，为社会富余资金和具有融资需求的实体经济单位提供便捷、高效、安全的投融资居间撮合服务。在积极引导社会富余资金帮助中小微企业健康发展的同时助推社会经济发展。

江苏紫金财富金融信息服务有限公司成立于2014年5月，总部座落在江苏省省会城市南京，目前在江苏省的南京、镇江、扬州、泰州、淮安、常州、盐城、连云港均设立了直营分公司，计划在2015年完成江苏省所有地级市及部分重点县级市的分公司覆盖。公司的核心团队由多年金融系统工作经验的行业精英和专业的互联网资深人士组成，在业务资源、项目质量、信用评估、风险核查、风险控制、风险化解、风控措施、数据安全等方面均参照专业银行标准严格把关，最大程度地保证投资人的投资安全。和融网是江苏紫金财富金融信息服务有限公司在线下金融服务成功运营的基础上，为投融资用户搭建的线上延伸服务平台。融资用户在通过和融网及平台合作机构的双重风控审核后，可以在和融网平台发布借款请求来实现个人或企业的快捷融资需要；投资用户可以把自己的富余资金通过和融网平台出借给信用良好有资金需求的个人或企业，在获得低风险，高收益的投资回报的同时帮助了优质的融资人或企业。与此同时亦促进了社会经济的健康发展，实现了多方共赢。
